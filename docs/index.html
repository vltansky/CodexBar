Homebrew: brew install --cask steipete/tap/codexbar

  • One status item per provider (Codex + Claude).
  • Session + weekly bars, plus reset times.
  • Optional OpenAI web access: Code review + usage breakdown.
  • Minimal UI, no Dock icon.

Codex

Prefers the local app-server RPC for limits + credits, with a PTY fallback to codex /status. Optional: reuse your browser session to show Code review + usage breakdown.

Claude Code

Runs /usage + /status locally and renders what the CLI reports (session + weekly).

Privacy

By default it reads local CLI output only. If you enable OpenAI web access, it reuses existing browser cookies (Safari/Chrome) to fetch the dashboard — no passwords stored.
